アップロード権限付与について

自分の裁量で、まあ大丈夫だろうという人に渡してください。

渡したあとでも権限剥奪出来るので、ダメそうだと思ったら剥奪してください。

権限を与えてはいけない人

  • 新参で誰かよくわからない
  • 名無しで人物がよくわからない
  • 荒らしっぽい人、悪用しそうな人
  • 違法画像、不快な画像を貼る人

権限付与方法

  1. https://sub-chat.onrender.com/permission を開く
  2. パスワードを入れる（一回入れたら次から保存される）
  3. 権限付与したい人の白トリを入れ、許可にして追加
  4. 権限付与したい人の拡張機能をONにさせてから、リロードさせる
  5. 権限付与したい人のチャット画面の上に赤い文字でアップロードできるようになりましたと出たらOK
  6. 許可リストにそのまま残してもいいが、白トリコロコロ変わる人は意味ないので削除しておく
  7. 画像は管理人オンリーのDiscordサーバーにアップロードされ、1分で削除されるがリンク自体は24時間有効、流出してもリスクがない画像にのみ使用することを説明する

追加オプション（やってもやんなくてもいい）

  • PCでは、ドラッグアンドドロップかコピーアンドペーストでアップロード出来ることを説明する
  • スマホでは、拡張設定のスマホモードをONにすると入力欄の左端に＋ボタンが出来るのでそこからアップロードできることを説明する
  • 暗号化ルームではDiscordのサーバーではなく管理者サーバーに暗号化してアップロードされ、内容は見えないことを説明する

権限剥奪方法

  1. https://sub-chat.onrender.com/permission を開く
  2. パスワードを入れる（一回入れたら次から保存される）
  3. 権限剥奪したい人の白トリを入れ、拒否にして追加
  4. 15分後使えなくなる
